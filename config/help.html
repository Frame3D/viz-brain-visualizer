Brain Visualizer Help

The Fl_Help_View widget displays HTML text. Most HTML 2.0 elements are supported, as well as a primitive implementation of tables.

Supported HTML tags: head, title, body (bgcolor/text/link), h1 through h6, p, ol, ul, li, dl, dt, dd, pre, center, table (border/bgcolor/colspan/align=left|center|right), tr, th, td, b, i, u, tt, font (color/size/face=helvetica|arial|sans|times|serif|symbol|courier), a (href/name), em, strong, code, kbd, var, br, hr, img (src/width/height/alt)

Supported color names: black, red, green, yellow, blue, magenta, fuchsia, cyan, aqua, white, gray, grey, lime, maroon, navy, olive, purple, silver, teal

Supported URL protocols: file, http, ftp, https, ipp, mailto, news

Supported image formats: GIF, JPEG, PNG
